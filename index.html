Geração de computadores

Primeira geração:

Havard Mark 1 (1944) era uma maquina de Calculo muito grande. Colossus (1946) criado pela Alemanha para fazer cripto analise em meados da segunda guerra, foi usado para Guerra. Eniac (1946) criado pelos EUA, este era mais automatizado.

Segunda Geracao (1959-1965): Geração marcada pelos transistores, já eramos computadores menores e mais "portateis" (portatil da epoca era poder mudar o computador de uma sala para outra).

Terceira Geração (1965-1970): Circuito integrado

  • Transistor
  • Capacitores
  • Resistores

Quarta Geração (a partir de 1971): Geração conhecida pelos computadores pessoais o que fez com que os computadores se tornassem mais populares e acessiveis.

Algumas pessoas acreditam que já estamos vivendo a 5ª geração com as Inteligencias artificiais, cripto moedas, computadores quanticos e com o meta verso.

Surgimento da Web

A internet é uma forma de comunicação.

O surgimento dos computadores e da internet andam em paralelo até que futuramente elas se encontram.

As primeiras formas de comunicação a partir de 1958 foram através de telegrafo, nesta epoca foi instalado cabos para interligares continentes.

Na década de 1950 com a Guerra Fria, nesta epoca com a corrida entre os dois paises por evolução foi criada a DARPA (Defense Advanced Research Projects Agency) que utilizada o computador Eniac.

Após muito tempo e evolução, surgiu a Arpanet para que locais distantes pudessem se comunicar. A partir da Arpanet foram criadas outras micro redes como por exemplo a MILNET voltada exclusivamente a area militar e a NSFNET voltada para noticias, tiveram tambem a rede NASA, CSNET, ESnet.

Os primeiros locais a se comunicarem no dia 29/10/1969 foi a Universidade da Californa e Instituto de pesquisa de Stanford.

Em 1989 surgiu o Tim Berners-Lee que viu a oportunidade de unir Hipertexto com TCP/IP para criar a World Wide Web (www). Sendo assim criado o primeiro navegador chamado WorldWideWeb.

Clients e Servers

(voltar)
Aqui veremos o que foi aprendido sobre Clients e Servers.

Clients

Clients é o lado que está consumindo conteudo da internet que vem do servidor.

Quando os clientes acessam a internet, ele solicita informações para o servidor que retorna arquivos HTML, CSS, Java Scrip, imagens, audios, entre outros arquivos. Esses arquivos são chamados de arquivos estáticos.

A velocidade de carregamento da pagina será diretamente afetada pela velocidade de internet que o cliente possui.

Quando o cliente faz um acesso a uma pagina de internet, o navergador gera um cache daquela pagina para que em acessos futuros, se o cache ainda estiver armazenado, o acesso fique mais rapido.

Servers

Os servidores são os computadores que fornecem as informações para os clientes.

Os servidores podem ser acessados através da internet ou podem ser servidores de rede locais, onde existe um computador central que armazena os arquivos.

Os servidores geralmente são computadores mais robustos por ter a necessidade de armazenar mais arquivos e tambem ser acessado por mais clients aos mesmo tempo.

Existem diferentes tipos de servidores como pode exemplo servidores de arquivos, segurança (Firewall), streaming, e-mail e Web.

Servidores de arquivos é um computador central que armazena imagens, audios, videos para que todos possam acessar.

Sergurança (Firewall) são sistemas que protegem redes, dados, sistemas, aplicações e dispositivos de TI contra ataques cibernéticos.

Servidores de Streaming são servidores dedicados a enviar dados para os clientes consumirem de acordo com a medida que for entrando informação.

Servidores de E-mails são responsaveis por armazenar os e-mails e fazer o envio deles.

Servidores Web são basicamente os servidores para a internet.

Tags

(voltar)
Aqui veremos o que foi aprendido sobre as Tags em HTML.

Toda tags HTML tem os sinais de "<", ">" e uma palavra entre os sinais. Quando colocamos o sinal de "/" significa que estamos fechando a tag.

Existem inumero tipos de tags, aqui veremos alguns exemplos como:

  • "input" que é uma tag para receber uma informação, podendo ser ela um texto, ou um numero, uma cor, entre outros tipos.
  • "img" está é serve para gerar imagens.
Algumas das tags não tem a necessidade de ser fechada, como por exemplo a tag "img".

Algumas tag possuem atributos, os atributos nada mais são do que propriedades que uma tag pode ter, alguns exemplos de atributos são:

  • id = está tag é um identificador.
  • style* = está tag serve para editar a forma do teste como cores, fonte.
  • class* = tag usada para padronizar textos.
  • type = Este é um atributo especifico da tag "input" para identificar qual tipo de informção será aceita pela tag.
* Tags que são utilizadas no CSS.

Links

(voltar)
Aqui veremos o que foi aprendido sobre Links em HTML.

Para podermos gerar links no nosso site para que seja possivel navegar nele e em outros links é necessario a utilização da tag "a" que significar "ancor".

Para que a tag "a" seja utilizada é necessario que seja atribuida a tag "href" que é a referência de para onde o link está enviando.

Existem 5 tipos de navegação com a tag "a" utilizando o atributo "target", são eles:

  • _blank = Utilizado para abrir em uma nova aba.
  • _self = Para o link seja aberto na mesma aba.
  • _top* = Enviaria o site para o frame do topo.
  • _parent* = Enviaria o site para o frame "pai".
  • # = Utilizada para ser ancorada a um id. Serve para que possa ir até um texto de forma mais rapida.
* Eram utilizada em uma época mais antiga onde havia mais de uma janela dentro da mesma aba
